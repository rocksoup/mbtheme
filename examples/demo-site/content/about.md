This is a demo site for the **Saunter** theme, an editorial-style Hugo theme built specifically for Micro.blog.

## About the Theme

Saunter combines clean typography, modern design patterns, and thoughtful features to create a beautiful blogging experience. Whether you're writing long-form articles, sharing quick thoughts, or curating links, Saunter adapts to your content.

## Features

- Editorial card layout
- Dark mode support
- Category organization
- Built-in search
- Newsletter integration
- Responsive design

## Getting Started

Visit the [GitHub repository](https://github.com/rocksoup/mbtheme) to learn more about installing and customizing the Saunter theme for your Micro.blog site.

## About This Demo

This demo site showcases the theme's key features with example posts including:

- Full articles with rich formatting
- Short micro posts
- Link posts
- Photography posts

Feel free to explore and see how the theme handles different types of content.
