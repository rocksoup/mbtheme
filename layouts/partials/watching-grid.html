How the Watching Page Works

This page now renders from the middleware-generated data feed (data/watched.enriched.json).

  1. Middleware resolves posters from TMDB and emits poster_url plus placeholder when missing.
  2. Hugo reads .Site.Data.watched.enriched.movies and renders this grid.
  3. New items appear after the scheduled enrich job updates the data file.

For local demos, drop a sample file into examples/demo-site/data/watched.enriched.json.
