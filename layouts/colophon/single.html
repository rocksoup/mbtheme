Colophon

System notes for Saunter — how the publishing stack fits together.

Purpose

Saunter is my personal link log. Pinboard bookmarks flow into Micro.blog via the microintegrations middleware, so this page captures what powers the site for future me (or another builder).

Platform & Hosting

  • Engine: Hugo compiled by Micro.blog with this Saunter theme bundled in the repo.
  • Hosting: Micro.blog serves the static site and handles uploads.
  • Automation: GitHub Actions in microintegrations sync Pinboard to Micro.blog every hour and commit updates.

Theme & Front-end

  • Theme: Custom Saunter theme (Fraunces for headings, Inter for body text, palette inspired by the Wonky Editorial prototype).
  • Components:
    • Timeline cards render full content with category badges.
    • The /seattle live cam pulls a fresh waterfront frame on each page load.
    • Light/dark toggle respects local time, OS preference, and manual overrides stored in localStorage.
  • Assets: Fonts from Google Fonts; icons are inline SVGs.

Newsletter & Forms

  • Provider: Mailchimp audience f981f7d258. Footer opt-ins post directly to Mailchimp with a honeypot field.
  • Search roadmap: /search embeds user-scoped search results so readers stay on-site while scanning the archives.

Middleware

  • Repository: github.com/rocksoup/microintegrations
  • Scripts: scripts/microblog-sync.mjs orchestrates Pinboard RSS → Micro.blog posts.
  • Env Vars: PINBOARD_MICROBLOG_FEED_URL, MICROBLOG_TOKEN, DRY_RUN.

Credits & Thanks

  • Micro.blog for the platform and Micro.blog API.
  • Pinboard for the simple RSS feed powering the integration.
  • The waterfront facing live cam embedded on /seattle.
  • Everyone filing issues in this repo—you shape the roadmap.

If you notice something broken or want to borrow ideas, open an issue or ping me on Micro.blog @jfunk.
